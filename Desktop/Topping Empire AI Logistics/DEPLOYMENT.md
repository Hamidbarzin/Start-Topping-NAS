# Deployment Guide (Render)

1. Go to [Render Dashboard](https://render.com)
2. Click **New → Web Service**
3. Connect your repo: `Hamidbarzin/Topping_Empire_AI-Logistics`
4. Set **Root Directory** = `.`
5. Set **Start Command** = `npm start`
6. Click **Create Web Service**
7. Wait for build to finish — your app will be live at:

https://topping-empire-ai-logistics.onrender.com

✅ Done — Node.js Express app deployed successfully!

**Note**: Render automatically installs dependencies and runs the start command. No build command needed.
